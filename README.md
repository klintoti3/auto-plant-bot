# Auto Tree Farm Bot

A Mineflayer-based Minecraft bot designed to keep your Aternos server online 24/7 while acting as an automated worker for your auto tree farms. You just give the bot saplings, and it will automatically plant them on nearby dirt blocks!

## ✨ Features
* **Auto-Planting:** Automatically detects dirt blocks nearby and plants saplings from its inventory.
* **Auto-Reconnect:** Will automatically attempt to rejoin the server if kicked or if the server restarts.
* **Aternos Compatible:** Works perfectly as an AFK bot to keep your free server online.
